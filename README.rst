Installation using ``pipenv``
=============================

1. Create a working directory

   .. code-block:: shell

		   $ mkdir my_project
		   $ cd my_project

2. Create a virtual environment

   .. code-block:: shell

		   $ pipenv install

3. Enter the virtual environment

   .. code-block:: shell

		   $ pipenv shell


4. Install ``ftntlib``

   .. code-block:: shell

		   $ git clone https://github.com/jpforcioli/ftntlib.git
		   $ cd ftntlib
		   $ python setup.py install
